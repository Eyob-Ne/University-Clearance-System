# University Clearance System

A web-based system designed to streamline and simplify the university clearance process for students and administrators. This system allows students to submit clearance requests, track approval status, and receive confirmation once all departments have cleared them.

 Features

* Student Registration & Login
* Department-Level Clearance Management
* Real-Time Clearance Status Tracking
* Admin Dashboard
* Secure Authentication
* Responsive and User-Friendly UI

 Tech Stack

* Frontend: React.js
* Backend: Node.js / Express.js
* Database: MongoDB 
* Version Control: Git & GitHub
